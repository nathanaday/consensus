
### MCP Server for IoT Data Streams

Extension and improvement on the `iot-sandbox` idea. This is an SDK to provide easy natural language interaction with time series data.


### Concept

Connect time series data streams and analyze them using natural language.

Easily add context to data points for better situational awareness.


## Phase 1

- Begin with time series data points
- Ingest data from memory, `csv`, or databases
- Support native db integration with Postgres and SQLite


### Phase 1 Skills

**Data Extraction**

Extract data from supported sources: `csv`, relational database


**Statistical Tools**

Support common time series data operations
